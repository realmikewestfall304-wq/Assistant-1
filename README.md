## Installation

1. Clone the repository:
```bash
git clone https://github.com/realmikewestfall304-wq/Assistant-1.git
cd Assistant-1
```

2. Run the assistant (requires Python 3.6+):
```bash
python3 assistant.py
```

## Usage

The assistant provides an interactive menu with the following options:

1. **Evaluate a business idea**: Get honest feedback on your business ideas
2. **Add a reminder**: Set reminders to keep you on task
3. **View reminders**: See all pending reminders
4. **Complete a reminder**: Mark a task as done
5. **Schedule an appointment**: Add appointments to your calendar
6. **View appointments**: See all scheduled appointments
7. **Draft an email**: Create email drafts
8. **View emails**: Review your email drafts and sent messages
9. **Get summary**: View a dashboard of all pending items
0. **Exit**: Close the assistant

## Data Storage

All your data is stored locally in `assistant_data.json` in the same directory. This includes:
- Reminders and their completion status
- Scheduled appointments
- Email drafts
- Business idea evaluations
