## Installation

No external dependencies required! Uses only Python standard library.

```bash
# Clone the repository
git clone https://github.com/realmikewestfall304-wq/Assistant-1.git
cd Assistant-1

# Run the assistant
python3 assistant.py
```

## Usage

Simply run the assistant and choose from the interactive menu:

```bash
python3 assistant.py
```

### Menu Options

1. **Add Task** - Create a new task with title, description, and priority
2. **List Tasks** - View all tasks or filter by status
3. **Complete Task** - Mark a task as completed
4. **Add Reminder** - Set a reminder with date/time
5. **List Reminders** - View all active reminders
6. **Check Emails** - View email summary
7. **Plan Workload** - Get automatic workload analysis
8. **Create Social Media Ad** - Design ads for social platforms
9. **List Social Media Ads** - View all created ads
10. **Add Appointment** - Schedule a new appointment
11. **List Appointments** - View all appointments
12. **Log Call** - Record a call with notes
13. **List Calls** - View call history
14. **Dashboard** - See complete overview
0. **Exit** - Close the assistant

## Data Storage

All data is automatically saved to `assistant_data.json` in the current directory. Your information persists between sessions.

## Example Workflow

```
1. Start the assistant
2. Add tasks for the day (option 1)
3. Check dashboard for overview (option 14)
4. Plan your workload (option 7)
5. Schedule appointments (option 10)
6. Set reminders (option 4)
7. Create social media ads (option 8)
8. Log calls (option 12)
9. Complete tasks as you finish them (option 3)
```
